##FekePHP  php5.4 FrameWork

FekePHPは、現在開発中のPHP5.4向けMVCフレームワークです。
データベースと簡単に連携でき、小中規模のアプリケーションを簡単に作成できるようになることを目標としています。

ドキュメント:http://fekephp.com/

### 動作環境
* PHP5.4以上
* MySQL,SQLite,PostgreSQL

### メイン機能
* ルーティング
* オートローダ
* O/Rマッパー(ActiveRecord風)
* テンプレートエンジン
* フォームHelper
* 画像加工
など
